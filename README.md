<h1>TODO LIST</h1>

<h4>Projeto de um todolist, criado com react native, ultilizando typescript, styled-components e asyncstorage para guaradas as informações salvas e alteradas</h4>

## Funcionalidades
- Criar tarefas
- Excluir tarefas
- Marcar tarefas como concluídas
